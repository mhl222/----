# samrt_albums_webfront

## 一、Project setup
```
npm install
```
### Compiles and hot-reloads for development
```
npm run serve
```
### Compiles and minifies for production
```
npm run build
```
## 二、简介

技术栈：` vue3 + vueX状态管理 + vue-router页面路由 +elementUI布局及相关组件` 

后端接口代码已完成：封装在net_util文件夹——index.js下待调用

项目进度：   仅剩前端逐步替换在线接口
